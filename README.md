## 📁 Repo Structure

```
hackforge/
├── README.md                          ← You are here
├── CONTRIBUTING.md                    ← How to log a new project
├── _template/
│   └── README.md                      ← Copy this for every new project
└── projects/
    └── YYYY-organizer-projectname/    ← One folder per hackathon
        ├── README.md
        ├── src/                       ← Source code (or submodule link)
        ├── assets/                    ← Screenshots, diagrams, demo GIFs
        └── docs/                      ← Architecture notes, pitch deck
```

### Naming Convention

```
projects/YYYY-organizer-projectname/
```

Examples:
- `2025-shekunj-intentflow`
- `2025-microsoft-xyz`
- `2026-acm-mlproject`
